安装MySql-8.0.11后连接navicat出错

刚开始学数据库，安装了mysql之后，不想用命令行，想用一下navicat，安装了navicat之后连接数据库报错，错误如下：

Client does not support authentication protocol requested by server; consider upgrading MySQL client

用了很多百度上的方法都没用，折腾了几天终于找到了解决的方法（虽然目前我也还不知道到具体的原因），但可以使用下面这条语句来解决：
ALTER USER ‘root’@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘831015’;
