以下是一些基本的命令（目前刚入门，很多其他的都还没用过）
npm install hexo -g &nbsp&nbsp&nbsp&nbsp&nbsp#安装
npm update hexo -g &nbsp&nbsp&nbsp&nbsp#升级
hexo init &nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp#初始化
命令简写：
hexo n “我的博客” == hexo new “我的博客” #新建文章
hexo g == hexo generate &nbsp&nbsp&nbsp&nbsp#生成
hexo s == hexo server &nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp#启动服务预览
hexo d == hexo deploy &nbsp&nbsp&nbsp&nbsp&nbsp #部署
hexo clean &nbsp&nbsp&nbsp&nbsp&nbsp&nbsp #清除缓存 网页正常情况下可以忽略此条命令
hexo g &nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp #生成静态网页
hexo d &nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp&nbsp#开始部署
