函数重载(overroad)：

函数名和返回值一样，但是参数的类型和个数不一样的两个或多个函数构成重载。调用方法时会根据参数列表的类型来选择对应的方法。

void max(int a,int b){
  System.out.println(a>b?a:b);
   }

void max(float a,float b){
 System.out.println(a>b?a:b);
  }

 void max(short a,short b){
  System.out.println(a>b?a:b);
  }
  //这三个函数构成重载，对于这三个函数的调用如下
  Test t=new Test;
  t.max(3,4)；           //调用参数为int型的函数
  t.max(3.1,4.1);         //调用参数为float型的函数
  short a=3,b=4;
  t.max(a,b);           //调用参数为short型的函数

函数重写（overwrite/override)

1.重写方法必须与被重写方法具有相同的方法名称，参数列表和返回类型
2.重写方法不能使用比被重写方法更严格的访问权限
