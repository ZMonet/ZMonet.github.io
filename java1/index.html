Java基础知识（1）

Java

内存分配：

局部变量的值存储在栈中，并且随函数的执行完毕自动消失，而通过new关键字创建的对象的值则存储在堆中，使用完后手动回收或由系统进行自动回收。

函数重载(overroad)：

函数名和返回值一样，但是参数的类型和个数不一样的两个或多个函数构成重载。调用方法时会根据参数列表的类型来选择对应的方法。

void max(int a,int b){
  System.out.println(a>b?a:b);
   }

void max(float a,float b){
 System.out.println(a>b?a:b);
  }

 void max(short a,short b){
  System.out.println(a>b?a:b);
  }
  //这三个函数构成重载，对于这三个函数的调用如下
  Test t=new Test;
  t.max(3,4)；           //调用参数为int型的函数
  t.max(3.1,4.1);         //调用参数为float型的函数
  short a=3,b=4;
  t.max(a,b);           //调用参数为short型的函数

函数重写（overwrite/override)

1.重写方法必须与被重写方法具有相同的方法名称，参数列表和返回类型
2.重写方法不能使用比被重写方法更严格的访问权限

Static关键字

在类中，用static定义的成员变量为静态成员变量，定义的方法为静态方法，

  1. 静态成员变量是该类的公有变量，第一次使用时被初始化，static成员变量只有一份

  2. 静态方法不再是针对于某个对象调用，所以静态方法不能访问非静态成员，但可以通过对象引用或者类名（不需要实例化）访问静态成员

This关键字

  1. 在类的方法中使用this关键字代表使用该方法的对象的“引用”
  2. 当必须指出当前使用方法的对象是谁时要使用this
  3. 使用this可以处理方法中成员变量与参数重名的情况，

Package和Import语句

  1. package语句必须放在Java源文件的第一行，指明该文件中定义的类所在的包（若没有该语句则指定为无名包）
